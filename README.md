# Video Encoder

Automatic video encoding pipeline written in Ruby.

## Features

- Queue-based encoding
- FFmpeg (NVENC)
- SQLite job repository
- Automatic verification
- Source archiving
- 1280×720 HEVC output

## Requirements

- Ruby 3.x
- FFmpeg with NVENC support
- SQLite3
